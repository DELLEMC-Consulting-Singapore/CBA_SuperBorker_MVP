# CBA_SuperBorker_MVP

## Service Catalog Setup

```bash
cd cba_ui/
```
Clone the repository and install the dependencies

```bash
npm install
```

Below command to bring service catalog UI

```bash
npm start
```


## OSB Setup

```bash
cd cba_broker_ui/
```
Clone the repository and install the dependencies

```bash
npm install
```

Below command to bring OSB UI

```bash
npm start
```
